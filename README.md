# JogoVelha_2bim

Jogo desenvolvido com Kotlin como linguagem de programação, Android Studio como ambiente de desenvolvimento. 
Jogo da velha simples, onde é possível colocar "peças" X ou O em turnos alternados, onde a "peça" X pode ganhar, ou a "peça" O, ou ainda dar empate, também chamado de velha, que é o nome do jogo.
